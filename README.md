# MistySoft QR-NFC App

Dashboard for QR-NFC SaaS system.

## Tech Stack

- React 18
- Vite
- React Router
- Tailwind CSS
- Lucide Icons
- Recharts (analytics)

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Deployment

Deploy to Vercel:
1. Connect GitHub repo
2. Configure environment variables
3. Deploy
